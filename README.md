# ⏸ Time to Pause

Windows desktop app that reminds you to take breaks and alerts you before Google Calendar calls.

![Version](https://img.shields.io/badge/version-1.4.0-black) ![Platform](https://img.shields.io/badge/platform-Windows-black)

---

## Features

- **Break reminders** — set interval (30 min, 1 hour, or custom) and get notified with sound, screen flash, or both
- **Google Calendar sync** — get alerted X minutes before your next call
- **Pixel-art UI** — clean black & white retro style
- **Runs in system tray** — stays out of your way

## Install

1. Download **[TimeToPauseSetup.exe](https://github.com/alexagnestor-wq/timetopause/releases/latest)**
2. Run it → follow the wizard (takes ~10 seconds)
3. Done — app starts automatically

> Windows may show a SmartScreen warning. Click **"More info" → "Run anyway"** to proceed.

## Usage

1. Choose notification type: **SOUND / FLASH / MIXED / RANDOM**
2. Choose interval: **30 MIN / 1 HOUR / CUSTOM**
3. Hit **▶ START**
4. Optionally click **📅 CONNECT CALENDAR** → sign into Google → get alerts before calls

## Custom Sounds

Drop any `.wav` file into `%APPDATA%\BreakReminder\assets\sounds\` to add your own sounds.

Special effects triggered automatically:
- `taste-the-rainbow-mf.wav` → pixel rainbow animation
- `alert.wav` → fullscreen flash with text
