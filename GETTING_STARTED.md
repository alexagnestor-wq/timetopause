## 🎯 Быстрый старт за 2 минуты

### Шаг 1: Запустите приложение
```bash
python main.py
```

Или дважды кликните на ярлык на рабочем столе.

### Шаг 2: Проверьте статус
Окно должно показать: **"Status: Stopped"** ✅

Это означает, что таймер НЕ запущен автоматически!

### Шаг 3: Выберите интервал
- Нужен быстрый тест? → Выберите **"⏱ 5 seconds (test)"**
- Нужен рабочий интервал? → Выберите **"⏱ 30 minutes"**

### Шаг 4: Выберите уведомление
Рекомендуем: **"🔊⚡ Sound + Flash"**

### Шаг 5: Нажмите START
Кнопка станет недоступна, кнопка STOP станет активной.

Статус изменится на: **"Status: Running (reminder every 5 sec)"**

### Шаг 6: Дождитесь напоминания
Через 5 секунд (или через выбранное время):
- 🔊 Вы услышите звуковой сигнал
- ⚡ Экран вспыхнет белым
- 💪 Встаньте и сделайте перерыв!

## 📝 Детальное описание каждого шага

### 1. Установка (первый раз)

```bash
# Откройте командную строку в папке TimeToPauseApp
# И выполните:

pip install -r requirements.txt
```

Необходимые библиотеки:
- `pygame` - для звуков
- `pystray` - для системного трея
- `Pillow` - для обработки изображений

### 2. Запуск через командную строку

```bash
# В папке проекта:
python main.py

# Вывод:
# ==================================================
# ⏸ Time to Pause Started
# ==================================================
# ✓ Click START button to begin
```

✅ Если увидели этот вывод - приложение запустилось правильно!

❌ Если вместо этого видите "Timer started" - это была старая версия, удалите конфиг:
```bash
python reset_config.py
```

### 3. Интерфейс приложения

```
┌─────────────────────────────────┐
│    ⏸ Time to Pause              │
│  Status: Stopped                │
├─────────────────────────────────┤
│ Interval between reminders:     │
│  ◉ ⏱ 5 seconds (test)           │
│  ○ ⏱ 30 minutes                 │
│  ○ ⏱ 1 hour                     │
│  ○ ⏱ 2 hours                    │
├─────────────────────────────────┤
│ Notification type:              │
│  ○ 🔊 Sound only                │
│  ○ ⚡ Flash only                │
│  ◉ 🔊⚡ Sound + Flash            │
├─────────────────────────────────┤
│  [▶ START]  [⏸ STOP]           │
│                                 │
│ Click ✕ to minimize to system   │
│ tray                            │
└─────────────────────────────────┘
```

### 4. Что означает каждый статус

| Статус | Значение |
|--------|----------|
| `Status: Stopped` | Таймер не работает, можете выбрать интервал |
| `Status: Running (reminder every 5 sec)` | Таймер работает, напоминание через 5 сек |
| `Status: Running (reminder every 30 min)` | Таймер работает, напоминание через 30 мин |

### 5. Управление из меню трея

Если вы минимизировали приложение в трей (нажали ✕):

1. **Найдите иконку "B"** в системном трее (справа внизу)
2. **Кликните правой кнопкой**
3. Вы увидите меню:
   - **🪟 Show Window** - показать главное окно
   - **▶ Start** - запустить таймер (если минимизировали)
   - **⏸ Stop** - остановить таймер
   - **⚙ Settings** - настройки (резерв)
   - **❌ Exit** - закрыть приложение полностью

## 🧪 Тестирование

### Быстрый тест (5 секунд)

```
1. Откройте приложение:
   python main.py

2. Проверьте в консоли:
   Должно быть: ⏸ Time to Pause Started
   НЕ должно быть: ✓ Timer started

3. Выберите: "⏱ 5 seconds (test)"

4. Выберите: "🔊⚡ Sound + Flash"

5. Нажмите: [▶ START]

6. В консоли появится:
   ✓ Timer started

7. Статус изменится на:
   Status: Running (reminder every 5 sec)

8. Подождите 5 секунд...

9. Произойдет:
   🔔 REMINDER! (14:35:42)
   - Белая вспышка на экране ⚡
   - Звуковой сигнал 🔊
   - Статус вернется на "Stopped"
```

✅ Если всё произошло как описано - всё работает правильно!

### Проверка конфига

```
1. Откройте файловый менеджер
2. Перейдите в: C:\Users\YOUR_NAME\AppData\Roaming\BreakReminder\
3. Откройте файл: config.json

Вы должны увидеть примерно:
{
  "interval_minutes": 0.08333333,
  "notification_type": "mixed",
  "sound_enabled": true,
  "flash_enabled": true,
  "volume": 100
}

Значение 0.08333333 = 5/60 минут = 5 секунд ✅
```

## 🐛 Если что-то не работает

### Проблема: "Timer started" в консоли при открытии

**Решение:**
```bash
# Удалите старый конфиг
python reset_config.py

# Запустите заново
python main.py
```

### Проблема: Нет звука

**Проверьте:**
1. Громкость в Windows включена
2. Файлы есть: `C:\Users\YOUR_NAME\AppData\Roaming\BreakReminder\assets\sounds\`
3. Файлы названы: `alert.wav` и `beep.wav`
4. Громкость в приложении = 100

### Проблема: Белая вспышка не появляется

**Проверьте:**
1. Выбрали "Sound + Flash" в интерфейсе
2. Монитор включен и работает
3. Видеодрайверы обновлены

### Проблема: Приложение вылетает при старте

**Выполните:**
```bash
# Проверьте версию Python
python --version  # Должно быть 3.7 или выше

# Переустановите зависимости
pip install -r requirements.txt --force-reinstall

# Удалите конфиг
python reset_config.py

# Попробуйте запустить
python main.py
```
